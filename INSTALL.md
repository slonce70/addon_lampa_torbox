# 📦 Інструкція по встановленню TorBox Enhanced Plugin

## 🎯 Швидке встановлення

### Для досвідчених користувачів:
1. Завантажте `torbox-lampa-plugin.js`
2. Додайте в Lampa через **Налаштування** → **Розширення**
3. Введіть API ключ TorBox
4. Готово! 🎉

---

## 📱 Детальні інструкції по платформах

### 🖥️ Windows (Desktop)

**Lampa Desktop App:**
1. Відкрийте Lampa
2. Натисніть `Ctrl + Shift + I` (відкрити DevTools)
3. Перейдіть на вкладку **Console**
4. Скопіюйте весь код з файлу `torbox-lampa-plugin.js`
5. Вставте в консоль і натисніть Enter
6. Плагін активується автоматично

**Через браузер:**
1. Відкрийте Lampa в браузері
2. Перейдіть в **Налаштування** → **Розширення** → **Додати розширення**
3. Виберіть "Завантажити з файлу" або "Вставити код"
4. Завантажте `torbox-lampa-plugin.js` або вставте код
5. Активуйте плагін

### 📱 Android

**Lampa Android App:**
1. Відкрийте Lampa
2. Перейдіть в **Налаштування** → **Розширення**
3. Натисніть **"+"** (Додати розширення)
4. Виберіть **"Завантажити з файлу"**
5. Знайдіть і виберіть файл `torbox-lampa-plugin.js`
6. Підтвердіть встановлення

**Через браузер Android:**
1. Відкрийте Lampa в Chrome/Firefox
2. Увімкніть **"Режим робочого столу"** в браузері
3. Слідуйте інструкціям для браузера (див. вище)

### 🍎 iOS/iPadOS

**Safari:**
1. Відкрийте Lampa в Safari
2. Натисніть **"AA"** в адресному рядку
3. Виберіть **"Запросити сайт робочого столу"**
4. Перейдіть в **Налаштування** → **Розширення**
5. Додайте плагін через **"Вставити код"**
6. Скопіюйте код з `torbox-lampa-plugin.js`

### 📺 Android TV / Smart TV

**Lampa TV App:**
1. Відкрийте Lampa на TV
2. Використайте пульт для навігації в **Налаштування**
3. Перейдіть в **Розширення** → **Додати**
4. Виберіть **"Ввести URL"** або **"Завантажити"**
5. Введіть посилання на плагін або завантажте з USB

**Через браузер TV:**
1. Відкрийте браузер на Smart TV
2. Перейдіть на сайт Lampa
3. Слідуйте інструкціям для браузера

## ⚙️ Налаштування після встановлення

### 1. Отримання API ключа TorBox

1. Перейдіть на [torbox.app](https://torbox.app)
2. Зареєструйтеся або увійдіть в акаунт
3. Перейдіть в **Settings** → **API**
4. Скопіюйте ваш **API Key**

### 2. Налаштування плагіна

1. В Lampa перейдіть в **Налаштування** → **Плагіни** → **TorBox Enhanced**
2. Вставте ваш API ключ
3. Налаштуйте параметри за потребою:
   - ✅ **Автоплей** - автоматично відтворювати перший файл
   - ✅ **Автовидалення** - видаляти торрент після перегляду
   - ✅ **Субтитри** - завантажувати субтитри автоматично
   - 🎬 **Якість** - оберіть бажану якість (1080p рекомендовано)
4. Збережіть налаштування

### 3. Перевірка роботи

1. Знайдіть будь-який фільм або серіал
2. Виберіть торрент джерело
3. Оберіть **"TorBox Enhanced"** як плеєр
4. Плагін повинен почати завантаження торрента

## 🚨 Усунення проблем встановлення

### Плагін не з'являється в списку

**Рішення:**
- Перезавантажте Lampa
- Перевірте консоль браузера на помилки (F12)
- Переконайтеся що код плагіна повністю скопійовано
- Спробуйте інший браузер

### Помилка "API Key invalid"

**Рішення:**
- Перевірте правильність API ключа
- Переконайтеся що акаунт TorBox активний
- Спробуйте згенерувати новий API ключ

### Плагін завантажується, але не працює

**Рішення:**
- Увімкніть режим налагодження в налаштуваннях
- Перевірте консоль браузера
- Переконайтеся що інтернет-з'єднання стабільне
- Спробуйте інший торрент

### Проблеми з відтворенням

**Рішення:**
- Перевірте налаштування плеєра Lampa
- Спробуйте інший формат відео
- Переконайтеся що торрент містить відео файли
- Збільште час очікування в налаштуваннях
